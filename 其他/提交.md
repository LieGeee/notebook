### 1. **初始化仓库（如果还没有初始化）**

```
git init
```

### 2. **添加文件到暂存区**

添加所有修改过的文件到暂存区：

```
git add .
```

或者，只添加特定文件：

```
git add <文件名>
```

### 3. **查看当前分支**

查看当前分支：

```
git branch
```

### 4. **提交更改**

将暂存区的文件提交到本地仓库：

```
git commit -m "提交信息"
```

### 5. **推送到远程仓库**

将本地仓库的内容推送到 GitHub 上的 `main` 分支（如果是 `master`，改成 `master`）：

```
git push -u origin main
```
